# SOCIONET Privacy

## Data Stored
- Account info, posts, media you create
- Messages (encrypted at rest)

## Never Sold
- No user data sold to advertisers
- No ad targeting based on messages

## Your Rights
- Export data: Settings → Privacy → Export
- Delete account: Settings → Delete Account
- Control visibility: Settings → Privacy

## Retention
- Deleted accounts: data removed within 30 days
- Logs: rotated after 7 days
